Date: 2026-04-30

## Executive Result

Bolt now pins all direct NautilusTrader git dependencies to upstream release
`v1.226.0`, peeled commit
`38b912a8b0fe14e4046773973ff46a3b798b1e3e`.

live-trading readiness claim and does not close the Polymarket CLOB V2 readiness

## Residual Risk

- The upstream support blocker is reduced, not closed. Bolt still needs
  live-readiness evidence for CLOB V2 signing/domain, pUSD collateral, order
  submission, fills, fee behavior, and reconciliation before live capital.
- The upstream release page labels `1.226.0` as Beta. This pin is acceptable for
  compile/test compatibility work, but any stable-release or capital-bearing
  claim must first review upstream errata and either re-approve this pin or bump
  to a later stable release.
- The prior `56a4382` audit remains useful history for the broader CLOB V2
  migration, but current source-of-truth docs must cite `v1.226.0`.
- Future NT pin changes must repeat this compatibility process rather than
  relying on green CI alone.
