# Proof Plan Adequacy Experiment: Selector Path

## Objective

Test whether the process can force the real late-discovered selector-path blocker classes into explicit pre-review claims and falsifiers.

## Corpus

- local selector-path code and tests in `src/clients/polymarket.rs`
- local raw-anchor fetch path in `src/platform/polymarket_catalog.rs`
- real review findings from selector-path PRs

## Important Constraint

This experiment is deliberately narrower than the whole process.

It tests proof-plan adequacy.
It does not pretend that proof-plan alone can solve stale review-target artifacts or scope-carried follow-up decisions.
